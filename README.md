# CollAI

CollAI 是一个面向“自主认知 AI 代理”的基础设施项目，目标不是单纯做一个 AI 编程助手，而是构建一个具备持续运行、长期记忆、自主学习与自我演化能力的认知系统。

## 项目现状

当前仓库以“项目设计与白皮书”为主，主要用于沉淀项目愿景、技术架构、发展路线和关键文档。后续将逐步补齐源码、测试和部署说明。

## 核心能力方向

- 自主认知循环
- 三层记忆系统
- 自我进化与代码修改
- 本地优先部署
- 多 Provider 弹性路由
- 安全与可观测性

## 发展路线

1. 完善代码仓库结构与模块划分
2. 把核心设计落地为可运行版本
3. 补齐测试、文档与部署流程
4. 逐步开放可用的后端/前端模块

## 贡献方式

欢迎对该方向感兴趣的开发者一起讨论、评审与实现。当前建议优先关注：

- 架构拆分与代码组织
- 后端服务骨架
- 前端界面与交互设计
- 测试与 CI/CD
